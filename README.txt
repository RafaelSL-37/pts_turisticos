pts_turisticos é uma aplicação desenvolvida como atividade para o curso "Criando poderosas API's RESTful com Django Rest Framework" da Udemy.
Link para aplicação no heroku: <https://pts-turisticos.herokuapp.com/> (29/12/2020)
Link para o curso: <https://www.udemy.com/course/apis-restful-com-django-rest-framework/> (29/12/2020)

O objetivo dessa aplicação é aprender a utilizar o framework Django-Restful.
Foi criado um banco de dados de pontos turísticos, com as tabelas descritas abaixo:
*pontos_turisticos (Tabela principal, contendo as informações de cada ponto turístico)
*atracoes (Tabela contendo as avaliações dos usuários para cada ponto turístico)
*avaliacoes (Tabela contendo as avaliações de cada ponto turístico)
*comentarios (Tabela com os comentários de cada ponto turístico)
*localizacao (Tabela com as informações da localização dos pontos turísticos)
*token (Tabela com os tokens de autenticação de cada usuário)
*user (Tabela com o registro dos usurários e suas permissões)
